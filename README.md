## Installation

### clone repo using ssh (recommended)

```sh
git clone git@github.com:niedzielnyaniol/todo.git
```

or using https:

```sh
git clone https://github.com/niedzielnyaniol/todo.git
```

```sh
cd todo
```

### Install dependencies

```sh
lerna bootstrap
```
